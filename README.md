# site-de-teste
html simples para demonstração da estrutura básica.
* podemos ver o uso do "href" para localizar os poemas gerado pela ChatGpt.
* "href" para o encaminhamento para sites de IA.
* uso da tag "select" para dar nota para cada poema.
* tags "h1", "H2" e "h3" com cores diferentes.

* TAGS USADAS:
* <header></header>
* <h1></h1>
* <h2></h2>
* <h3></h3>
* <p></p>
* <pre></pre>
* <select>
* <hr>
* <ul></ul>
